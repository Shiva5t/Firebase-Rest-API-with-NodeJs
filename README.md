# Firebase-Rest-API-with-NodeJs


**GET ALL USERS**
----
  This API gets all the users from Firebase database

* **URL**

  /api/GetAllUsers

* **Method:**

  `GET`
  

* **Data Params**

    None

* **Success Response:**
  

  * **Code:** 200 <br />
    **Content:** `{ id : 12, name: "Abc", age: 14 }`
 
* **Error Response:**

 
  * **Code:** 500 <br />
    **Content:** `{ error : "Issue with getting response." }`
